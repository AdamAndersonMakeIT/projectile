    Play

    Helpers for traced Play Framework controllers and views (API Documentation)

    Usage

    sbt
    libraryDependencies += "com.kyleu" %% "projectile-lib-play" % "latest"
    Maven
    <dependency>
  <groupId>com.kyleu</groupId>
  <artifactId>projectile-lib-play_2.12</artifactId>
  <version>latest</version>
</dependency>
    Gradle
    dependencies {
  compile group: 'com.kyleu', name: 'projectile-lib-play_2.12', version: 'latest'
}
